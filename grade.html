Grade

Letter Done? Description
D Successfully wrangle the data into a hierarchical format that can be interpreted by either d3.hierarchy() or d3.stratify(). The hierarchy must have height 3 or more (i.e. at least 3 levels including the root and leaf nodes) and each leaf node must have an attribute.
C Complete the D-level functionality and implement a node link visualization using the d3.cluster() or d3.tree() layout. You may use any Coordinate system (Cartesian or polar) and any edge style (straight or curved).
The node attribute must be encoded somehow (i.e. using color, size, shape, etc.). Nodes must have a minimum details-on-demand interactivity showing the node label.
B Complete the C-level functionality and implement a space-filling or enclosure visualization using d3.partition(), d3.treemap(), or d3.pack() layouts. You may use any Coordinate system (Cartesian or polar).
The node attribute must be encoded somehow (i.e. using color, size, shape, etc.). Nodes must have a minimum details-on-demand interactivity showing the node label.
A Complete the B-level functionality and implement filtering, brushing, or zooming in either of the two hierarchical visualizations you created. You only need to implement this interactivity in one of the visualizations, but must still support details-on-demand interactivity in that visualization.
