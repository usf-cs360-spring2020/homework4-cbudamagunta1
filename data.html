Fire Department Calls for Service

The Fire Department Calls-For-Service dataset records all fire units responses to calls. They track all relevant time intervals, location information, call type, and disposition.

The data was last accessed on April 18, 2020 and is updated daily

Processing

This assignment necessitated forming a hierarcy, which involved some wrangling of the data.

  • Filter to only include calls in 2019
  • Group by Call Type > Neighborhoods > Call Final Disposition
  • Roll-Up by Incident Count
  • Sort by Call Type

The wrangling reduced the dataset to 2,542 rows, and included these columns:

  • Call Type: The specific type of call made.
  • Incident Number: A unique 8-digit number assigned by DEM to this Fire incident.
  • Call Final Disposition: Disposition of the call (Code). For example TH2: Transport to Hospital - Code 2, FIR: Resolved by Fire Department
  • Neighborhoods - Analysis Boundaries: Neighborhood District associated with this address
